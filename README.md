# React Rich Text Editor Tab Integration
An example demonstration of the EJ2 React Rich Text Editor with the Syncfusion Tab component integration

## Quick start

Prerequisites:

- Node.js 20.x (recommended) and Also supported: Node.js 22.x or 24.x
- npm

Install dependencies and run the app:

```bash
npm install
ng run dev
```

Open localhost link in your browser.

## What this sample shows

- How to include the Rich Text Editor in an React component
- How to integrate the Tab component in the Rich Text Editor
